# Function/Method Names

```
//before - after

quizToQuestion - getQuestionFromQuiz            // получение строки вопроса из объекта викторины
claimName - getClaimName                        // получение имени клейма из JWT
get - contains                                  // проверка содержания значения в коллекции 
getScoreForQuiz - calculateScoreFromQuizzes     // вычеслвение объекта "Счет" из списка викторин
run - runQuiz                                   // процедура запуска викторины раннером викторины
read - readInputFromResource                    // чтение входящей текстовой информации из ресурса 
write - writeOutputToResource                   // запись исходещей текстовой информации в ресурс
writeError - writeErrorToResource               // запись исходящей текстовой информации в ресурс
getAuthor - getAuthorById                       // получение автора по id
count - countAllBooks                           // подсчет всех книг в сервисе книжного магазина
update - updateBook                             // обновление данных книги в сервисе книжного магазина 
create - createBook                             // создание новой книги в сервисе книжного магазина  
getCounterStatistics - getBookStoreStatistic    // получение показателей статистики в сервисе книжного магазина
```
