# Variable names
## Примеры имён в коде, которые следует избегать
```
//before - after

temp - referenceSlot                        // опорный слот в структуре данных

result - arithmeticResult                   // результат арифметичекского вычесления

sum - nodesValuesSum                        // сумма значений узлов связанного списка

record - parsedResourceChunk                // элемент записи данных из разных источников данных

sum - hashCode                              // переменная хранящая результат вычесления хэша

averageOrderAmount -  orderAmountAverage    // средняя сумма заказа

value - userInput                           // значение введенное пользователем

index - slotIndex                           // индекс слота в труктуре данных

list - users                                // коллекция пользователей

time - connectionlifeTimeS                  // время жизни соединения

node1 - firstNode                           // первый элемент коллекции

data - serializedData                       // массив байт сериализованных данных
```
